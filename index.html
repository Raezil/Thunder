Why Choose Thunder?

🚀

gRPC + REST + GraphQL

Automatically expose RESTful APIs and GraphQL endpoints from gRPC services with gRPC-Gateway and grpc-graphql-gateway. Triple protocol support with minimal configuration.

🗄️

Prisma Integration

Efficient database management and migrations with type-safe queries. Support for PostgreSQL, MySQL, and SQLite with automatic schema generation.

☸️

Kubernetes Ready

Deploy and scale effortlessly with built-in Kubernetes support, service discovery, and automatic load balancing for cloud-native applications.

🔒

TLS Security

Secure gRPC communications with TLS encryption, JWT authentication, and rate limiting middleware out of the box for enterprise-grade security.

📊

Structured Logging

Built-in Zap logging with structured output for better observability and debugging in production environments with real-time monitoring.

🔗

GraphQL Gateway

Seamlessly expose your gRPC services as GraphQL endpoints with grpc-graphql-gateway. Enable modern frontend development with flexible queries and real-time subscriptions.

Modular & Extensible

Easily extend Thunder for custom use cases with a clean, modular architecture that scales with your needs.

🎯

Multi-Protocol Support

Serve your APIs through gRPC, REST, and GraphQL simultaneously. Perfect for teams with diverse client requirements and modern development workflows.

Quick Start

# Install Thunder CLI
git clone https://github.com/Raezil/Thunder.git
cd Thunder
chmod +x install.sh
./install.sh

# Create a new project
thunder init myapp
cd myapp
go mod tidy
# Define your service (example.proto)
syntax = "proto3";
package example;

import "google/api/annotations.proto";
import "graphql.proto";

service Example {
  rpc SayHello(HelloRequest) returns (HelloResponse) {
    option (google.api.http).get = "/v1/example/sayhello";
    option (graphql.schema) = { type: QUERY, name: "sayhello" };
  }
}

message HelloRequest {
  string name = 1 [(graphql.field) = { required: true }];
}

message HelloResponse {
  string message = 1;
}

# Generate REST and GraphQL endpoints
thunder generate --proto=example.proto --enable-graphql
go run ./cmd/app/server/main.go

# Server accessible at:
# gRPC: localhost:50051
# REST: localhost:8080
# GraphQL: localhost:8080/graphql
